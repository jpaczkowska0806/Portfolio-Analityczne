# Portfolio Analityczne i Programistyczne

Repozytorium zawiera wybrane projekty zrealizowane w ramach studiów na kierunku Modelowanie Matematyczne i Analiza Danych na Uniwersytecie Gdańskim. Skupiają się one na zastosowaniu metod statystycznych, algorytmiki oraz inżynierii danych.

## 📊 Analiza Danych i Statystyka (R)
* **Eksploracyjna analiza danych (EDA):** Badanie demografii i zachowań finansowych z wykorzystaniem ankiet (`projekt1.R`, `projekt1.pdf`).
* **Segmentacja i Analiza Giełdowa:** Analiza wskaźników Cena/Zysk dla 246 spółek oraz badanie dynamiki szeregów czasowych (`projekt2.R`, `projekt3.R`). Pełne raporty z wnioskami znajdują się w dołączonych plikach PDF.

## 🐍 Algorytmy i Metody Numeryczne (Python / SageMath)
* **Algorytmy i Struktury Danych:** Parser wyrażeń algebraicznych ONP (`ONP.py`) oraz implementacja klasycznych algorytmów sortujących wraz z testami wydajnościowymi (`algorytmy_sortujace.py`).
* **Metody Numeryczne:** Samodzielna implementacja iteracyjnej metody Gaussa-Seidla do rozwiązywania układów równań liniowych (`Gauss-Seidl kod.pdf`, `Gauss-Seidl.pdf`).
